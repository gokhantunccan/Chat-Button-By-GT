Add chat button to your wordpress site with Chat Button By GT Plugin.

== Description ==

With this plugin, you can add a chat button to your website. You can customize the chat button you added.

Key features:

*   Adding and customizing WhatsApp chat button
*   Adding and customizing Telegram chat button
